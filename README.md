# 文中排班工具

文中 eHRS 人資系統的桌面輔助工具，支援排班查看/編輯、打卡異常檢視、班別建議。

---

## 安裝方式（同事請看這裡）

### 第一步：下載安裝檔

點擊下方連結，下載 `安裝文中排班工具.command`：

👉 **[點我下載安裝檔](https://github.com/pony9632-pixel/ehrs-tool/releases/latest/download/安裝文中排班工具.command)**

### 第二步：執行安裝

1. 在下載的檔案上按右鍵 → **「打開」**（第一次需要這樣才能執行）
2. 跳出確認視窗時點「打開」
3. 等待安裝完成（約 1 分鐘）

> 如果出現「無法打開，因為無法驗證開發者」：  
> 前往「系統設定 → 隱私權與安全性」→ 點「仍要打開」

### 第三步：啟動

安裝完成後，桌面會出現「**文中排班工具**」資料夾，  
之後直接雙擊裡面的 `啟動文中排班工具.command` 即可。

---

## 功能

| 分頁 | 說明 |
|------|------|
| 排班 | 查看/編輯整月排班，雙擊格子可快速改班 |
| 打卡 | 查詢打卡明細，自動標示遲到/早退/忘打卡（紅字） |
| 建議 | 依實際打卡推薦最適班別，支援加班建議 |
| 設定 | 填入 eHRS 帳號密碼（存在本機，不上傳） |

## 自動更新

程式每次啟動會自動檢查 GitHub 是否有新版本，有的話自動下載更新並重啟，不需手動操作。

---

## 系統需求

- macOS 10.15 以上
- Python 3.9 以上（[下載](https://www.python.org/downloads/macos/)）
